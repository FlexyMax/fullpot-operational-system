# Contexto de MiMo - FullPot Operational System

## Stack
- **Framework**: Next.js 16.1.6 (App Router) + React 19 + TypeScript
- **Estilo**: Tailwind CSS 4, Radix UI, Lucide icons, Framer Motion
- **State**: Zustand (localStorage) + TanStack React Query
- **Backend**: Next.js API Routes → SQL Server (mssql)
- **Auth**: NextAuth v4 (credentials)
- **Deploy**: Vercel

## Bases de Datos
- **FULLPOT** — datos operacionales (invoices, inventory, customers, etc.)
- **SISTEMA** — configuración, usuarios, permisos, pantallas

## Estructura de Directorios

## Router del Menú (`src/app/menu/page.tsx`)
- `getRoute(appPage)` mapea `app_page` (de DB) → ruta Next.js
- `getIcon(appPage)` mapea a ícono Lucide
- Los permisos vienen de SP `sp_sis_modules_screens_by_user`
- Estructura: `{ [moduleName]: [{ app_page, app_title, app_icon }] }`
